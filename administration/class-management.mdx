Classes are the building blocks of how your school is organized in Fleedu. Every student belongs to a class, every teacher is assigned to one or more classes, and activities, attendance, and daily reports all flow through the class structure. Getting your classes set up correctly at the start of the year — with the right name, age group, capacity, and teacher — means everything else in Fleedu will fall into place automatically. This guide walks you through creating classes, populating them with students, and keeping them organized throughout the year.

## Creating a New Class

<Steps>
  <Step title="Go to Class Management">
    From your Fleedu dashboard, navigate to **Administration → Classes**. You will see a list of all existing classes. Click the **+ New Class** button in the top-right corner to open the class creation form.
  </Step>

  <Step title="Fill In the Class Details">
    Complete the following fields to define your new class:

    | Field | Description | Example |
    |---|---|---|
    | **Class Name** | A clear, recognizable name for the group | Tulip Class, Bintang A, Kelompok B1 |
    | **Age Group** | The intended age range for children in this class | 3–4 years, 4–5 years, 5–6 years |
    | **Capacity** | Maximum number of students the class can hold | 15, 20 |
    | **Assigned Teacher** | The primary teacher responsible for this class | Select from your staff list |
    | **Co-Teacher (optional)** | A second teacher or teaching assistant | Select from your staff list |
    | **Description (optional)** | Internal notes about the class curriculum or schedule | Full-day program, Monday–Friday |

    <Tip>
      Use a consistent naming convention from the start — for example, naming classes after flowers, colors, or planets — so parents and teachers can easily identify them in reports, notifications, and billing documents.
    </Tip>
  </Step>

  <Step title="Save the Class">
    Click **Create Class**. The class now appears on your Classes page and is ready to receive students. The assigned teacher will see it in their Fleedu account immediately.
  </Step>
</Steps>

## Assigning Students to a Class

Once a student is on your school roster (either enrolled through the online form or added manually), you can assign them to a class in two ways:

<CardGroup cols={2}>
  <Card title="From the Student Profile" icon="user-graduate">
    Open **Administration → Students**, click the student's name, and select **Assign to Class** from their profile. Choose the class from the dropdown and confirm.
  </Card>
  <Card title="From the Class Roster" icon="chalkboard-user">
    Open **Administration → Classes**, click the class name, then click **Add Students**. Search for students by name and add one or multiple at once. This is the fastest method when setting up a class for the first time.
  </Card>
</CardGroup>

<Note>
  A student can only be assigned to **one class at a time**. If you add a student to a new class, they will be automatically removed from their previous class. Their historical records (attendance, reports, activities) from the previous class are preserved and remain accessible.
</Note>

## Viewing a Class Roster

To see all students currently enrolled in a class, navigate to **Administration → Classes** and click the class name. The **Roster** tab shows:

- Each student's name, photo, and date of birth
- Enrollment date in this class
- Parent contact details (tap to open)
- Attendance summary for the current week

From the roster view you can also quickly export the student list as a PDF — useful for printing a physical attendance sheet or sharing the list with a substitute teacher.

## Editing Class Details

To update a class name, capacity, age group, or assigned teacher at any time:

1. Go to **Administration → Classes** and click the class you want to edit.
2. Select the **Settings** tab.
3. Make your changes and click **Save Changes**.

<Warning>
  Changing the assigned teacher on a class will immediately update that teacher's access. The previously assigned teacher will no longer see the class or its students in their Fleedu account unless they are added as a co-teacher. Make sure to communicate this change to your staff.
</Warning>

## Moving Students Between Classes

You may need to move a child mid-year due to age progression, a change in schedule, or a class size adjustment.

1. Open the student's profile from **Administration → Students**.
2. Under **Class Assignment**, click **Change Class**.
3. Select the destination class from the dropdown.
4. Confirm the move.

<Info>
  The move is logged with a timestamp in the student's profile history. If your school tracks promotions at the end of each academic year, this history gives you a clear record of which class a child was in at any given time.
</Info>

## Academic Years and Class Archiving

At the end of each academic year, you can **archive** classes you no longer need for day-to-day operations. Archived classes:

- No longer appear in the active class list
- Are not available for new student assignments
- **Retain all historical data**, including attendance, daily reports, and activity records, which remain fully accessible for reference and auditing

<Accordion title="How do I archive a class?">
  Go to **Administration → Classes**, open the class Settings tab, scroll to the bottom, and click **Archive Class**. Confirm the action. You can view archived classes at any time by toggling the **Show Archived** filter on the Classes page.
</Accordion>

<Accordion title="Can I unarchive a class?">
  Yes. Open the archived class from the Classes page (with Show Archived enabled), go to Settings, and click **Restore Class**. This makes the class active again and available for new assignments.
</Accordion>

<Accordion title="Should I create new classes each year or reuse existing ones?">
  We recommend creating new classes at the start of each academic year. This keeps your current-year data clean and ensures that reports, billing cycles, and attendance records are clearly separated by year. Archive last year's classes rather than deleting them so the historical record is always available.
</Accordion>

<Accordion title="What happens to students in an archived class?">
  Students in an archived class are not deleted — their profiles remain active on the roster. However, they will appear as **Unassigned** until you move them into a new active class. Use the **Bulk Assign** feature during year-transition setup to move multiple students at once.
</Accordion>
